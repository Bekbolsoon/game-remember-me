# vue3-game

## Project setup
In order to create or edit a game, you need to download:
* Vue CLI
```
npm install -g @vue/cli
yarn globak add @vue/cli
```
* Create vue project in current folder
```
vue create .
```

## Lauch of the project

* Compiles and hot-reloads for development
```
npm run serve
```
* Compiles and minifies for production
```
npm run build
```
* Lints and fixes files
```
npm run lint
```

## Project logic
public > index.html > div#app
src > main.js > import VUE and APP && mount to div#app

App - elements before board game
BoardVue - board without board item, button and level
BoardItem - board item
